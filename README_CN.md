# Cursor 规则项目

本项目是个人收集的 Cursor 使用规则，分为两类：

1. **全局规则（`.md` 文件）**：适用于所有项目的通用指南。
2. **项目规则（`.mdc` 文件）**：针对特定项目，可以覆盖或补充全局规则。
